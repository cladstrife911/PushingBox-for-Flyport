Project description:

The application shows how to use PushingBox services.
There is an external library called pushingbox.c
To illustrate the use of pushingbox services a grove potentiometer and a RGB LED are used.
To be able to run this application you need to be registered on http://www.pushingbox.com/
PushingBox is a very simple and intuitive tool to push notifications (mail, Android, Karotz, and more...).
